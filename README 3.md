# Proyecto QA: Simulación de escenarios de prueba utilizando Chrome DevTools

## Descripción

Este proyecto se enfocó en el uso de Chrome DevTools para modificar respuestas y simular distintos escenarios de prueba sin necesidad de realizar cambios en el backend. El objetivo fue validar el comportamiento de la interfaz ante diferentes condiciones de datos.

## Objetivos

* Aprender el uso de Chrome DevTools para pruebas.
* Simular respuestas del servidor.
* Validar el manejo de errores.
* Verificar el comportamiento de la interfaz.

## Mi rol

QA Engineer Junior

Responsabilidades:

* Configuración de DevTools.
* Modificación de respuestas.
* Ejecución de pruebas exploratorias.
* Registro de hallazgos.
* Documentación de resultados.

## Herramientas utilizadas

* Chrome DevTools
* Local Overrides
* Network Inspector
* Navegador Chrome

## Escenarios evaluados

### Simulación de datos válidos

* Respuestas exitosas.
* Datos completos.

### Simulación de errores

* Campos vacíos.
* Datos inconsistentes.
* Errores de servidor.
* Respuestas incompletas.
