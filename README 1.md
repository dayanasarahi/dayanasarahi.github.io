# Proyecto QA: Validación de las funcionalidades "Método de Pago" y "Agregar Tarjeta"

## Descripción

Este proyecto consiste en la validación funcional de las ventanas "Método de Pago" y "Agregar Tarjeta" dentro de una aplicación web. El objetivo fue verificar que los usuarios puedan registrar, visualizar y seleccionar métodos de pago de manera correcta, garantizando una experiencia segura y libre de errores.

## Objetivos

* Analizar los requisitos funcionales.
* Diseñar una lista de comprobación (checklist).
* Crear casos de prueba positivos y negativos.
* Aplicar técnicas de análisis de valores límite.
* Detectar posibles defectos y riesgos de la funcionalidad.

## Alcance de las pruebas

### Método de Pago

* Visualización de métodos disponibles.
* Selección de tarjetas registradas.
* Validación de datos mostrados.

### Agregar Tarjeta

* Registro de nuevas tarjetas.
* Validación de campos obligatorios.
* Manejo de errores.
* Restricciones de formato.

## Casos evaluados

### Casos positivos

* Registro exitoso de una tarjeta válida.
* Selección correcta de un método de pago existente.
* Visualización adecuada de la información.

### Casos negativos

* Campos vacíos.
* Número de tarjeta inválido.
* Fecha de expiración incorrecta.
* Código de seguridad inválido.
